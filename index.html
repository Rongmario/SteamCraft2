Steamcraft 2 - MC v1.6.4

A mod that extends Minecraft by adding post-apocalyptic, retro-futuristic blocks/items.
You are welcome to contribute to this mod, however, you will need to discuss things with me beforehand. Additionally, you will need to use the proper java code conventions in order to contribute.
These conventions can be located here: http://www.oracle.com/technetwork/java/codeconv-138413.html
